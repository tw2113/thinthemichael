== Installation ==

= Manual Installation =

1. Upload the InstaGo folder to the plugins directory in your WordPress installation
2. Activate the plugin.
3. Navigate to the "Settings > InstaGo Settings" Menu.

Upload Zip File

1. Navigate to Plugins -> Add New.
2. Click the "Upload Plugin" button.
3. Select the "instago.zip" file from your computer.
4. Click "Install Now".
5. Click the "Activate Plugin" link.
6. Navigate to the "Settings > InstaGo Settings" Menu.
