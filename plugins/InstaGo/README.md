## Description ##

Dynamic Instagram profile URLs that can be managed from WordPress

## Installation ##

### Manual Installation ###

1. Upload the entire `/instago` directory to the `/wp-content/plugins/` directory.
2. Activate InstaGo through the 'Plugins' menu in WordPress.
